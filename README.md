Wire
====================================================================================
Blender - rendering in wireframe.

====================================================================================
Installation

User Preferences - Addons - Install from file
    
    WireS.py - Shows render process
    or
    WireH.py - Hide render process

====================================================================================
Usage

Ctrl + Shift + F12 in 3D view window
